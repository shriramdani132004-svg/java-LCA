# Java LCA Project – Internship & Exam System

This project is a Java-based web application developed as part of the LCA submission.
It includes features for managing internships, conducting exams, and viewing results.

---

## 📥 How to Access the Project

1. Open this repository
2. Click on **Code → Download ZIP**
3. Extract (unzip) the downloaded file
4. Open the project folder in your IDE (Eclipse/IntelliJ)

---

## ▶️ Running the Project

* Import the project into your IDE
* Configure Apache Tomcat server
* Set up MySQL database using provided SQL file
* Run the project on server

---

## 🛠 Technologies Used

* Java (JSP & Servlets)
* MySQL
* HTML, CSS
* Apache Tomcat

---

## 📌 Note

All source code files are included in this repository.
